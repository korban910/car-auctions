### Preparation

Install [minikube](https://minikube.sigs.k8s.io/docs/).

Install [kubectl](https://kubernetes.io/docs/reference/kubectl/).

Alternativaly, use `brew` to install:

```
brew install minikube
brew install kubectl
```
